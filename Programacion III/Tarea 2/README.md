En el siguiente trabajo, se puede encontrar una lista enlazada, en la cual se encuentran 5 programas. Cada vez que el usuario selecciona una opción del menú y se ejecuta una función recursiva, el resultado se guarda como un nuevo nodo dentro de la lista enlazada.
De esta manera el programa mantiene un historial de todas las operaciones realizadas y puede visualizarlos cuando selecciona la opcion de ver lista enlazada.
